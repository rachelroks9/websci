How to Query:

  1. Keep the query box with the default search. Press "Run Query." This will find 10 schools and its campus and average class size. Results are sorted by campus size.
  2. Enter your own SPARQL query into the query box and select "Run Query."

Please Keep In Mind:

  • If your query does not seem to load (i.e. the loading circle is still visible) then there is an error in your query. Please check the error console to see exactly what is wrong with your query.
  • It is recommended that you refresh the page in between all queries.
